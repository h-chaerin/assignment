# PostgreSQL
## 관리 도구


- 서버 도구
	- postgres
		- 최상위 서버 데몬
	- pg_ctl
		- 명령어
			- 서버 시작
			- 서버 중지
			- 서버 상태

	- initdb
		- 초기화 명령어
	- pg_resetxlog
		- 트랜잭션 로그 초기화 명령어

- 클라이언트 도구
	- psql
		- 기초적인 관리 툴
		- 명령어 기반 인터페이스
		- 자동 완성
			- 셀과 유사

		- 스크립트 통한 자동화 기능

	- pgAdmin
		- 그래픽 사용자 인터페이스
		- 6.3.2부터 지원
	- phpPgAdmin
		- 웹 기반의 관리 툴
		- PHP로 작성
